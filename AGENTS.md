# AGENTS.md

## Project

AshyqAI landing page for `ashyqai.kz`.

## Publishing

This project is published through GitHub Pages. Keep the site compatible with static hosting from the repository root unless the project owner explicitly changes the publishing setup.

## Development Notes

- The main landing page is `index.html`.
- Prefer plain HTML and CSS for this project.
- Do not add a required build step, backend service, package manager dependency, or runtime dependency unless explicitly requested.
- Keep assets and links compatible with GitHub Pages static hosting.
- Use `support@ashyqai.kz` as the default public support email unless instructed otherwise.
